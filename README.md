# Tradesman's Playbook — Pinterest Pins

Pin images for Pinterest. Format: 1000x1500px PNG.

## Folder structure

```
electrician/    ← Electrician Business board pins
hvac/           ← HVAC Business board pins
plumber/        ← Plumbing Business board pins
roofer/         ← Roofer Business board pins
handyman/       ← Handyman Business board pins
generic/        ← Contractor Resources + Trade Tips board pins
```

## Raw URL format

```
https://raw.githubusercontent.com/plocicmilan/tradesmansplaybook-pins/main/{trade}/pin_{NNN}.png
```

## Usage

Generated by `tradesman-playbook/prirucnik/generate_pins.py`
Uploaded to Pinterest via API (`tradesman-playbook/pinterest/pin_uploader.py`)
